## Features

### Board Structure
- Columns with titles
- Tasks with titles and descriptions
- Horizontal rows and vertical stacks
- Layout presets and column sizing controls

### Editing
- Inline editor (Markdown)
- Inline editor (WYSIWYG)
- Overlay editor (Markdown)
- Overlay editor (dual Markdown + preview)
- Overlay editor (WYSIWYG)
- Undo/redo
- VS Code keybindings/snippet insertion
- WYSIWYG toolbar for text styles, links, embeds, blocks, and tables
- Context menus for links, embeds, and diagrams

### Top Bar
- File name
- YAML header settings (Marp, Pandoc)
- Card/column drop sources
- Column folding and header stickiness
- Sorting controls
- Active processes indicator
- Save state
- Layout templates
- Main menu

### Drag & Drop
- Task and column reordering
- Drop sources for tasks and columns
- External file drops
- Clipboard-based drops
- File import dialogue

### File Import Dialogue
- Link existing file
- Copy file to Media folder
- Search for file by filename
- Open Media folder
- Cancel

### Media & Diagrams
- Image, video, audio, and PDF embeds
- Draw.io and Excalidraw embeds
- Mermaid and PlantUML diagrams
- Diagram previews and editing
- Path conversion and replacement tools

### Hash Database (MediaIndex)
- Duplicate detection for dropped files
- Configurable scan scope (media folders, content folders, workspaces)

### Includes & Links
- Markdown includes (!!!include)
- Includes in columns, tasks, and task content
- Wiki links

### Tags
- Tag parsing and display
- Tag colors and categories
- Tag visibility controls

### Export
- Marp export
- Pandoc export
- YAML-driven export options

### Templates
- Column templates

### Search & Broken File Handling
- Broken link indicators
- Search/browse for replacement files
